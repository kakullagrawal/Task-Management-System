To-Do List Manager ✅

A simple Python-based To-Do List Manager that helps users organize and track their daily tasks. The application provides a menu-driven interface for adding, viewing, marking tasks as completed, and deleting tasks. All tasks are stored in a text file, allowing data to persist between sessions.

Features
Add new tasks
View all tasks
Mark tasks as completed
Delete tasks
Save tasks to a file
Load tasks automatically when the program starts
Technologies Used
Python
File Handling
Lists and Dictionaries
Functions
How to Run
Install Python on your system.
Clone or download this repository.
Open a terminal in the project directory.
Run the program:
python todo_list.py
Functionalities
Add Task

Create and store new tasks in the task list.

View Tasks

Display all tasks along with their completion status.

Mark Task as Done

Update a task's status to completed.

Delete Task

Remove tasks that are no longer needed.

Persistent Storage

Tasks are saved to a tasks.txt file and automatically loaded when the application starts.
